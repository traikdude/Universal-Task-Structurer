# 🛡️ Security Policy

## Supported Versions

We actively support the latest version of the Universal Task Structurer.

| Version | Supported |
| ------- | --------- |
| v1.7.x  | ✅ Yes    |
| < v1.7  | ❌ No     |

## Reporting a Vulnerability

If you find a security vulnerability, please do not open a public issue. Instead, report it privately to `traikdude@gmail.com`.

We will acknowledge your report within 48 hours and coordinate a fix release within 14 days.
